# Heart Disease Classification using Machine Learning

## Problem Statement
Predict whether a patient has heart disease using machine learning classification algorithms.

## Dataset
- Name: Heart Disease Dataset
- Source: Kaggle
- Instances: 1000+
- Features: 13
- Target: Presence of heart disease (0 = No, 1 = Yes)

## Algorithms Used
1. Logistic Regression
2. Decision Tree Classifier
3. K-Nearest Neighbor
4. Naive Bayes
5. Random Forest
6. XGBoost

## Project Workflow
- Data loading
- Data preprocessing
- Feature and target separation
- Train-test split
- Model training
- Accuracy comparison
